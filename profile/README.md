# PODAAC

## PODAAC Tools and Services

Most of the PODAAC tools and services are open source and managed in this podaac github organization.

A significant part of the service proposed by PODAAC are integrated in the EOSDIS architecture and contribute to the [EarthData](https://www.earthdata.nasa.gov/) portal:
- for the user community through [harmony](https://harmony.earthdata.nasa.gov/) middleware
- for data upstream processes through [cumulus](https://www.earthdata.nasa.gov/news/feature-articles/earth-science-data-cloud-eosdis-cumulus-project) workflow engine 

In addition, some standalone services are specifically operated by PODAAC:
- [hydrocron](https://github.com/podaac/hydrocron): specific API to access river flow data from SWOT
- [swodlr](https://github.com/podaac/swodlr): SWOT On-Demand Level 2 Raster Generation
- [feature translation service](https://github.com/podaac/feature-translation-service): for API SWOT data subsetting
- [confluence](https://github.com/podaac/confluence)
- [soto](https://github.com/podaac/SOTO) (portal soon to retire)
- [hitide](https://github.com/podaac/hitide) (portal soon to retire)
- [mcc](https://github.com/podaac/mcc): metadata validation tool for data providers
- [generate](https://github.com/podaac/generate): L2P product generation from OB.DAAC
- [hitide-backfill-tool](https://github.com/podaac/hitide-backfill-tool): to generate footprint and quicklook on cumulus-missed granule
- Virtualized DataSet, soo to come
